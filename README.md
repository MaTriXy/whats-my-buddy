# whats-my-buddy

A fun scratch card that reveals your Claude Code buddy before you hatch it.

## What Is This?

Claude Code introduced a buddy system -- small companions that sit beside your input box and comment in a speech bubble. This project is a simple interactive scratch card (`index.html`) that lets you scratch and reveal what buddy you'd get, before hatching it inside your Claude Code terminal.

Open `index.html` in your browser, scratch the card, and see your buddy.
